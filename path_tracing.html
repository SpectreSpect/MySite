About the project

This project was writed with c++ language. It was very important to use GPU, because without it rendering such expencive (about preference) images would take too much time. For this task I used library DirectX 11, wich is very often used in computer games.

Making this project take a lot of time. It took me about six months to create this. In fact, it took so long due to the fact that when starting this project, I did not know с++ programming language almost at all and the directX library, and even more so I did not know how path tracing works.

But I am completely satisfied with the result.
